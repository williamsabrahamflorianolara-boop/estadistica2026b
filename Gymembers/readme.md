# About the Dataset

This dataset provides a detailed overview of gym members' exercise routines, physical attributes, and fitness metrics. It contains **973 gym member records**, including key performance indicators such as heart rate, calories burned, and workout duration.

Each record also includes demographic information and experience levels, enabling comprehensive analysis of fitness patterns, athlete progression, and health trends.

## Dataset Features

| Feature | Description |
|---------|-------------|
| **Age** | Age of the gym member. |
| **Gender** | Gender of the gym member (*Male* or *Female*). |
| **Weight (kg)** | Member's weight in kilograms. |
| **Height (m)** | Member's height in meters. |
| **Max_BPM** | Maximum heart rate (beats per minute) during workout sessions. |
| **Avg_BPM** | Average heart rate during workout sessions. |
| **Resting_BPM** | Resting heart rate before the workout. |
| **Session_Duration (hours)** | Duration of each workout session in hours. |
| **Calories_Burned** | Total calories burned during each workout session. |
| **Workout_Type** | Type of workout performed (e.g., Cardio, Strength, Yoga, HIIT). |
| **Fat_Percentage** | Body fat percentage of the member. |
| **Water_Intake (liters)** | Daily water intake during workouts, measured in liters. |
| **Workout_Frequency (days/week)** | Number of workout sessions completed per week. |
| **Experience_Level** | Fitness experience level, ranging from **1 (Beginner)** to **3 (Expert)**. |
| **BMI** | Body Mass Index (BMI), calculated from height and weight. |

## Potential Applications

This dataset is ideal for:

- 📊 Data science and machine learning projects.
- 🏃 Fitness and exercise behavior analysis.
- ❤️ Health and physiological research.
- 📈 Modeling fitness progression over time.
- 🔍 Exploring relationships between demographic, physiological, and performance variables.

Its diverse set of variables makes it possible to analyze how different factors influence workout intensity, endurance, calorie expenditure, and overall health outcomes.
